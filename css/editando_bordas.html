Configurações de bordas

ARREDONDANDO OS VÉRTICES

border-radius:(valorGeral)px ; OU border-radius:(valor1)px (Valor2)px (valor3)px (valor4)px;

A ordem das margem e caso queira fazer individualmente: border-top-left-radius:"valor"px;. Na ordem fica, top-left -> top-right -> bottom-right -> bottom-left.

Também funciona em %

Borda personalizada

Configure a borda do jeito que quiser (precisa ter uma borda de apoio), depois, inserir o comando "border-image-source: url('link da borda personalizada.png');".

Pode ser, que precise fatiar a imagem -> border-image-slice:"valor teste";. Não precisa colocar px.

Também pode brincar com: border-image-repeat: repeat;. Pode usar o repeat, stretch...

shorthand

border-image:;. Ordem: source slice repeat.
