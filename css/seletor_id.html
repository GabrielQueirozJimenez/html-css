Basics

A tag <span> é usada apenas para "separar" um treçho do texto

Normal

É possivel uma tag ter mais de uma classe, apenas coloque o nome das duas ou mais classes, sem virgula, sem problemas. Ex: class="basics advanced"

Advanced
