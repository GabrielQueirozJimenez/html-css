HEADER

A tag <header>, significa cabeçalho.

MAIN

A tag <main>, onde fica o conteudo principal.

SECTION

A tag <section>, sessão/sessões.

ARTICLE

A tag <article>, artigo/artigos.

ASIDE

A tag <aside>, conteudo periferico/relativo ao artigo.

FOOTER

A tag <footer>, é o rodapé

NAV

A tag <nav>, é boa para criar menus. Nav -> Navegação.
