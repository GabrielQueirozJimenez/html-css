Modelo de caixa

box, básicamente o conteúdo e mede por "height" e "width".

padding, espaço entre a borda e o conteúdo, mede por (top, bottom, right, left).

border, onde o próximo conteúdo gruda se não houver uma margem.

outline

margin, é onde o próximo conteúdo vai "grudar".

Há dois tipos de caixas

box-level

Ocupa a linha inteira e pula para a outra, para criar outra box.

  • <div>
  • <h1>-<h6>
  • <p>
  • <main>
  • <header>
  • <nav>
  • <article>
  • <aside>
  • <footer>
  • <form>
  • <video>

inline-box

Cria uma box na mesma linha, tentando otimizar o máximo de espaço.

  • <span>
  • <at>
  • <code>
  • <small>
  • <strong>
  • <em>
  • <sup> - <sub>
  • <label>
  • <button>
  • <input>
  • <select>
