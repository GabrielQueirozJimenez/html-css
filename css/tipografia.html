Tipografia

Anatomia do tipo

Serifadas

Fonte serifadas têm uma extensão (um traço) no final de algumas letras.

Importancia: Ela cria no subconsciênte, uma linha imaginaria que ajuda a guiar na leitura. (Foi criada principalmente para o papel)

Sans-Serif

Não têm serifas.

Importante: Para textos longos em sites, use fontes sem ferifas. (CUIDADO com fontes sem muitos espaços.)

Monoespaçadas

Fontes monoespaçadas, possuem o mesmo espaço para todos os caracteres.

Handwriting

Simula textos escritos à mão.

Display

Em suma, é o resto. São fontes comemorativas.
