Tabelas

Tabelas possuem as tags e a hierarquia <table> -> <tr> -> <td>.
Também tem a <th>

tr = table row, td = table data

Para melhorar a qualidade da tabela, também temos as tags <thead>, <tfoot>, <tbody>

Servem para cabeçalho, rodapé e corpo, respectivamente.
