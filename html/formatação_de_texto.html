Principais formatações de texto

Negrito e destaque

O comando <b> serve para deixar as coisas em negrito.

O commando <stronge> serve para destacar algo.

Itálico ou Ênfase

O comando <i> serve para deixar em itálico.

O comando <em> serve para dar enfâse.

Texto Marcado

Para marcar um texto, pode usar a tag <mark>.

Texto grande e pequeno

Pode usar a tag <small> para diminuir o texto.

A tag <big> é usada para aumentar, mas está obsoleta (Usar CSS).

Texto deletado e inserido

Para sublinhar um texto use a tag <u>, mas está obsoleta (Olhar)

Para marcar um texto como inserido, a tag <ins> é usado

Para marcar um texto como excluído, use a tag <del>

Texto sobrescrito e subscrito

Use a tag <sup> para deixar o texto sobrescrito.

Use a tag <sub> para deixar o texto subscrito.
