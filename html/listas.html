Listas Ordenadas

Tag <ol> para fazer uma lista ordenada, onde a ordem FAZ diferença. Pode escolher o tipo, com o type, e por onde começa com start

  1. Primeiro
  2. Segundo
  3. ...

Listas não Ordenadas

Tag <ul> para fazer uma lista não ordenada, onde a ordem NÃO importa.

  • Neiko
  • Saala
  • Lily

Lista Mista

  1. Parte 1-A
    • Alguma coisa
    • Outra coisa
    • Coisa
  2. Parte 1-B

    2. Tanto faz
    3. Algo para fazer
