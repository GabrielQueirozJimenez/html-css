Trabalhando com links

Links Externos

Use a tag <a> para criar um link externo.
E use target="_blank" para acessar o link, em uma nova aba.
E o rel="external", para fazer a semântica de um link externo.

Links Internos

Usando a mesma, tag <a>, você pode direcionar o usuário para uma página que esteja na mesma pasta.

Nesse caso, rel="next" e/ou rel="prev" e o target="_self" ou nem coloca...

Links para download

Para criar um link de download, use a tag <a>, selecione o arquivo para download, pode usar também <a href="ItemParaDownload" dowload="ItemParaDownload"></a>

Caso não dê: <a href="ItemParaDownload" dowload="ItemParaDownload" type="application/pdf"></a>

Ou use <a href="ItemParaDownload" target="_blank"></a>

Quando criar uma âncora, <a href="#"></a> para dizer, que é um link vázio
