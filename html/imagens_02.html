Imagens dinâmicas

Use a tag <picture> para configurar uma imagem dinâmica.

A tag <picture>, permite que sejam criados vários sources dentro.
Os sources, normalmente vêm com "min-width", e usar uns 50px a mais, que é o tamanho aproximado da barra de rolagem.
Ps. Usar os sources em ordem crescente ou decrescente.
